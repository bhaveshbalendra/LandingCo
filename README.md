## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+

### Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>

   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```
